## Repository with a simple example with Lambda + Gateway

Using CDK for Terraform by Hashicorp.

### How to deploy
```shell
cd ./src
cdktf deploy
```

### How to destroy
```shell
cd ./src
cdktf destroy
```

### How to see a diff
```shell
cd ./src
cdktf diff
```
